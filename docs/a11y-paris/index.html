Confort+  🟧

Une solution open-source pour améliorer le confort de tous et mettre en valeur les sites web accessibles !

  • Plus de 20 réglages disponibles.
  • Organisés en 9 modes d’usage.
  • Disponible sur Firefox, Edge et Chrome.

Une expérimentation perpétuelle  🔬

Des équipes pluridisciplinaires et une collaboration spécifiques.

  1. Les équipes XDLab d’Orange, sous l’égide de Denis Chêne, pour l’ergonomie et la conception fonctionnelle.
  2. Une équipe technique de Onepoint pour expérimenter et implémenter les réglages.
  3. Divers intervenants chez Orange : Loïc Laussel pour les aspects open-source, EASE pour l’accessibilité, des testeurs…

Tout est open-source, sur GitHub : sources, version, documentation, backlog, etc.

Pourquoi une extension  🔌

Une extension navigateur peut être considérée comme une technologie d’assistance.

  • Elle est propre au navigateur de l’utilisateur, contrairement aux overlays.
  • Elle peut donc servir sur tous les sites, contrairement à une version auto-hébergée.
  • Les réglages sont conservés dans le navigateur, et pourraient même être synchronisés entre plusieurs navigateurs.
  • Elle étend le navigateur sans promettre des miracles : si le site visité est inaccessible, l’extension ne pourra pas œuvrer convenablement.

Installation  🔽

Installez l’extension depuis le dépôt GitHub Orange-OpenSource/Orange-Confort-plus ou téléchargez directement l’extension :

  • Pour Firefox :
    1. rendez-vous sur about:debugging#/runtime/this-firefox,
    2. « Charger un module complémentaire temporaire »
    3. et sélectionnez l’extension que vous venez de télécharger.
  • Pour Edge et Chrome :
    1. extrayez le .zip téléchargé,
    2. puis allez dans « Gérer les extensions »,
    3. « Charger l’extension décompressée »
    4. et sélectionnez le répertoire extrait.

Pensez à épingler l’extension à votre barre personnelle si ça n’est pas fait automatiquement.

Visite guidée  🦮

Sur confort-plus.orange.com, pour démarrer !

  1. Activez l’extension en cliquant sur .
  2. Lancez Confort+ en ouvrant .
  3. Sélectionnez un mode d’usage.
  4. Activez et modifiez des réglages.
  5. Vous pouvez à tout moment mettre Confort+ en pause.
