Crédits

Ce site utilise les polices de caractère suivantes :

  •  Bello Pro — créée par Underware;
  • Museo Slab — créée par Exljbris.

Ces typographies ont été choisies grâce à Typecast et sont servies par Typekit.

La coloration syntaxique dans les articles se sert de prismjs créé notamment par Lea Verou.

Ce site est fièrement propulsé par WordPress, enjolivé avec le thème ffeeeedd — ma propre création, sous licence CC BY 3.0 FR. — librement adapté.

Le thème lui-même est basé sur une adaptation libre des micro-frameworks CSS suivants :

  • KNACSS — créé par Raphaël Goetter;
  • RÖCSSTI — créé par Nicolas Hoffmann.

Ces deux solutions sont sous licence CC-BY.

Des crédits plus détaillés sont mentionnés dans la documentation et les fichiers du thème sur GitHub.
